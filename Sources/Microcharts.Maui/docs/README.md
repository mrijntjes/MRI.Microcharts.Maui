Lemathieu.Microcharts.Maui

This is a fork of the Microcharts repository and updated for .net 9 and SkiaSharp 3.x

This is specific done to compliance for the new 16KB page compatibility for Android.

Depends on SkiaSharp 3.119.1 (is 16KB page compatible!)
NuGet package

Search for the following package: Lemathieu.Microcharts.Maui
Or use manualy by attaching project

    Add the Microcharts.Maui project (Not the Microcharts base project!) to your App project

    Go to your App project and add a project reference. The Microcharts.Maui project is shown.
